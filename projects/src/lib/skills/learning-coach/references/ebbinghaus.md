# 艾宾浩斯遗忘曲线

## 核心发现

德国心理学家艾宾浩斯发现：**遗忘是有规律的**

## 遗忘曲线

```
记忆保持率
100% |●
 80% |●
 60% |●●
 40% |   ●●
 20% |      ●●●
  0% |__________●●●●
     1  2  3  7  30 天
```

**关键数据**：
- 20分钟后：遗忘42%
- 1小时后：遗忘56%
- 1天后：遗忘74%
- 1周后：遗忘77%
- 1个月后：遗忘79%

## 间隔复习策略

### 最佳复习时间点

| 复习节点 | 时间间隔 | 记忆保持率 | 复习后保持率 |
|----------|----------|------------|--------------|
| 第1次 | 学习后1天 | ~26% | ~60% |
| 第2次 | 学习后3天 | ~40% | ~75% |
| 第3次 | 学习后7天 | ~23% | ~85% |
| 第4次 | 学习后30天 | ~15% | ~95% |

### 复习方法

**不是重读**，而是**主动回忆**：

1. 看到主题名称
2. 闭上眼睛回忆内容
3. 检查记忆是否准确
4. 补充遗忘的部分

## 在本技能中的应用

### 日程创建逻辑

```python
def create_review_schedule(topic, learn_date):
    intervals = [1, 3, 7, 30]  # 天
    for days in intervals:
        review_date = learn_date + timedelta(days=days)
        create_calendar_event(
            title=f"复习提醒: {topic}",
            date=review_date,
            description="请回顾概念卡，检查记忆情况"
        )
```

### 记忆状态追踪

| 状态 | 判定条件 | 提示 |
|------|----------|------|
| 🟢 良好 | 7天内复习过 | 继续保持 |
| 🟡 注意 | 7-14天未复习 | 建议复习 |
| 🔴 危险 | 超过30天未复习 | 高遗忘风险 |

## 科学依据

1. **间隔效应**：分散学习比集中学习效果更好
2. **提取练习**：主动回忆比重读更有效
3. **元认知监控**：知道"自己知道"比"实际知道"更重要
