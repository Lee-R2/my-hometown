# 费曼学习法

## 核心理念

> "如果你不能简单地解释它，你就没有真正理解它。" —— 理查德·费曼

## 四步法

### 第一步：选择一个概念
明确你要学习的知识点

### 第二步：用简单语言解释它
假装你在教一个10岁的孩子：
- 不要用专业术语
- 用大白话说明白
- 如果卡住了，说明你没真懂

### 第三步：发现漏洞并补充
解释不清楚的地方，就是你的知识盲区：
- 回去查资料
- 重新理解
- 再尝试解释

### 第四步：简化与类比
- 能用一句话说清吗？
- 能找到生活中的类比吗？
- 能画个图说明吗？

## 引导提问示例

| 阶段 | 提问 |
|------|------|
| 解释阶段 | "你能用大白话跟我说说这个概念吗？" |
| 深入阶段 | "那它能解决什么实际问题？" |
| 类比阶段 | "有没有生活中类似的例子？" |
| 边界阶段 | "什么情况下它不适用？" |
| 关联阶段 | "它和你之前学的XX有什么关系？" |

## 为什么有效

1. **主动输出** > 被动输入
   - 听讲记住10%
   - 思考记住50%
   - 讲解记住90%

2. **暴露盲区**
   - 讲不出来的地方就是没懂的
   - 强迫你正视知识漏洞

3. **建立连接**
   - 用类比连接已知知识
   - 形成知识网络而非孤岛
