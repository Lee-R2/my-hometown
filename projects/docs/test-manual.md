## 六、重点观察项

在所有测试过程中，请特别关注：

1. **错误提示是否友好**：是否显示中文提示而非英文错误码或堆栈
2. **Toast 通知是否正常**：成功/失败操作是否有 toast 弹出
3. **控制台是否有报错**：F12 Console 中是否有红色错误
4. **API 响应格式**：在 Network 面板中查看 API 返回，是否为 `{ error, code }` 格式
5. **页面是否白屏**：任何操作后页面不应白屏
